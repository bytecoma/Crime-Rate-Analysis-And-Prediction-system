CRIME   RATE
ANALYSIS AND
PREDICTION   SYSTEM

"This system is a data driven analytical platform design to visualize and predict crime trends across Indian states. It provides the users an interactive dashboard. Users here can select the years, states, and types of crime to view corresponding statics on a data driven map. Additionally, a charts based module allows users to compare crime trends across multiple years and states using bar-charts, line-charts, nested-pie-charts, pie-charts. The system emphasis accessibility, interpretability and visualizalion making it a powerful tool for public awareness and policy level insights."
